# 📝 Note Flow

A modern full-stack note management application built with **Next.js 16**, **React Query**, **Zustand**, and a clean **Neon UI**.

> Organize your ideas, tasks, work notes and important information in one place.

---

## ✨ Features

- 🔐 Authentication (Sign up / Sign in / Logout)
- 👤 User profile
- 📝 Create notes
- 📚 Browse notes by category
- 🔎 Live search with debounce
- 🗂 Filter notes by tags
- 🗑 Delete notes
- 📄 View note details
- 📱 Fully responsive (Desktop, Tablet & Mobile)
- ⚡ Smooth loading experience
- 🌙 Modern Neon Dark UI

---

## 🛠 Tech Stack

### Frontend

- Next.js 16 (App Router)
- React 19
- TypeScript
- CSS Modules
- Zustand

### Backend

- REST API
- JWT Authentication

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/JenyaLinx/noteflow.git
```

Go to the project

```bash
cd noteflow
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
